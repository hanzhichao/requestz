# Requestz

基于pycurl封装的http请求包

# 特性
- [x] dns解析时间/连接时间/下载时间等更多的响应统计信息
- [x] 响应对象自带find支持xpath/jsonpath/re
- [x] 响应及其中字段支持jsonschema验证
- [x] 自带一个runner
